# Call

## ポインタ
|No|ポインタ               |意味|
|-:|-----------------------|----|
| 1|pc (プログラムカウンタ)| |
| 2|sp (スタックポインタ)  | |
| 3|bp (ベースポインタ)    | |

## プログラム

|No|項目                   |値|
|-:|-----------------------|--|
| 1|呼び出す関数           |P |
| 2|呼び出す関数のアドレス |p |
| 3|引数の数               |a |
| 4|ローカル変数の数       |c |

```
func P(A0, A1, ... , Aa-1)
    var L0, L1, ... , Lc-1;
    .....
end
```

## 呼び出し

## 引数のpush

a個の引数をpushした後、スタックは以下のようになる。

|アドレス|値    |変数|ポインタ|
|-------:|-----:|----|--------|
|     100|      |    |        |
|     101|  a0  |A0  |        |
|     102|  a1  |A1  |        |
|     103|...   |... |        |
|   100+a|  aa-1|Aa-1|        |
|   101+a|      |    |←sp    |

## CALL p

`CALL p`を実行する。

`CALL p`は以下を行う

```
PUSH pc   # このときpcはCALL pの次のアドレスを指している。
PUSH bp
pc = p
bp = sp
```

|アドレス  |値    |変数|ポインタ|
|----------|-----:|----|--------|
|     100  |      |    |        |
|     101  |  a0  |A0  |        |
|     102  |  a1  |A1  |        |
|     ...  |...   |... |        |
|   100+a  |a(a-1)|Aa-1|        |
|   100+a+1|旧pc  |    |        |
|   100+a+2|旧bp  |    |        |
|   100+a+3|      |    |←sp, bp|

## ローカル変数初期化

ローカル変数`L0, L1, ... , Lc-1`の初期値を
`v0, v1, ... , v(c-1)`とすると、
以下を実行してローカル変数領域を初期化する。

```
PUSH v0
PUSH v1
.....
PUSH v(c-1)
```

|アドレス   |値    |変数|ポインタ|
|-----------|-----:|----|--------|
|100        |      |    |        |
|101        |  a0  |A0  |        |
|102        |  a1  |A1  |        |
|...        |...   |... |        |
|100+a      |a(a-1)|Aa-1|        |
|100+a+1    |旧pc  |    |        |
|100+a+2    |旧bp  |    |        |
|100+a+3    |  v0  |L0  |←bp    |
|100+a+4    |  v1  |L1  |        |
|...        |...   |... |        |
|100+a+c+2  |v(c-1)|Lc-1|        |
|100+a+c+3  |      |    |←sp    |

## 関数Pの実行

関数P内に定義された文を実行する。
このとき引数、ローカル変数は以下のようにbpを使ってアクセスする。

|No|変数の分類  |変数|アドレス           |
|-:|------------|----|-------------------|
|1 |引数        |A0  |bp-a-2             |
|2 |            |A1  |bp-a-2+1           |
|3 |            |... |...                |
|4 |            |Aa-1|bp-a-2+a-1 = bp - 3|
|5 |ローカル    |L0  |bp                 |
|6 |            |L1  |bp+1               |
|7 |            |... |...                |
|8 |            |Lc-1|bp+c-1             |

引数、ローカル変数のアドレスの一般式は以下のようになる。

|No|変数|アドレス           |
|-:|----|-------------------|
|1 |Ai  |bp - a - 2 + i     |
|2 |Li  |bp + i             |

## Return

関数から呼び出し元に復帰するには以下を実行する。

```
sp = bp
bp = POP
pc = POP
sp = sp - a
```
