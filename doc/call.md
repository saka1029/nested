# Call

## ポインタ
|No|ポインタ               |意味|
|-:|-----------------------|----|
| 1|pc (プログラムカウンタ)| |
| 2|sp (スタックポインタ)  | |
| 3|ap (引数ポインタ)      | |
| 4|lp (ローカルポインタ)  | |

## プログラム

Fの先頭アドレスは「8」

```
func F(a, b, c)
    var x = 7, y = 8;
    .....
end
```
# 呼び出し

```
F(1, 2, 3)
```

# 命令
CALL命令実行まで。
|No|実行命令|        |
|-:|--------|--------|
| 1|LIT 1   |        |
| 2|LIT 2   |        |
| 3|LIT 3   |        |
| 4|CALL 8  |8はFの先頭アドレス|
